DESC Weak Lensing Deblending Package
====================================

Fast simulations and analysis for the Weak Lensing Working Group of the LSST `Dark Energy Science Collaboration <http://www.lsst-desc.org>`_.

This software was primarily developed to study the effects of overlapping sources on shear estimation using the Fisher Formalism. 

The code is hosted on `github <https://github.com/ismael2395/WeakLensingFisherFormalism>`_.  Please use the `issue tracker <https://github.com/ismael2395/WeakLensingFisherFormalism/issues>`_ to let us know about any issues you have with installing or running this code, or to request new features.
